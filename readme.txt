Приложение использует API Яндекc-Словаря - https://yandex.ru/dev/dictionary/
Перед сборкой проекта нужно получить API ключ и указать его в файле api.properties, в корне проекта.
Принцип работы приложения - добавление английских слов.
Если добавляемое слово существует в онлайн-словаре яндекса, то слово сохраняется с дополнительной информацией (перевод, часть речи, транскрипция),
иначе слово добавляется без дополнительной информации. Во втором случае, в дополнительную информацию можно добавить собственные перевод и часть речи.

*Приложение разрабатывалось с целью получения навыков использования шаблона MVVM, библиотек room и retrofit.
